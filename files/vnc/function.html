vnc.gemfield.org有哪些特别的功能?

1，各种IDE，已经安装的有Qt Creator, IDEA；

2，Java构建工具maven, gradle等;

3，Dolphin，你见过的最强大的文件管理器，统一的视角去看本地和远端server和windows办公电脑上的文件，并且可以直接diff；

4，samba服务，/u/windows可以map到你的windows办公电脑上，成为和c盘d盘并列的一个盘；

5，终端中可以F1/F5一键登录build server；

6，gitk等Git图形化工具；

7，在任何有互联网的地方可以方便接入而无需vpn等设置；

8，kompare这个强大的diff前端GUI界面；

9，firefox不用设置可以访问google；

10, Kafka broker, Kafka client, Schema registry.

11, docker
